# Notion笔记页面 同步 HexoNexT博客文章

# 能够做什么

将Notion笔记页面及嵌套的子页面同步到Hexo博客中

# 如何使用
1. 将需要同步的Notion页面导出为markdown文件压缩包
2. 在部署本项目后，会提供文件上传接口，将压缩包上传后，会生成博客文章到预先定好的博客目录下
